# OmniCalc 🧮

> **OmniCalc** — заманауи әрі көпфункционалды математикалық мобильді қосымша. Flutter технологиясымен жасалған, Android және iOS платформаларында жұмыс істейді.

## 📱 Мүмкіндіктері

OmniCalc келесі негізгі функцияларды ұсынады:

| Иконка | Мүмкіндік | Сипаттама |
|--------|-----------|-----------|
| 🏠 | **Басты бет** | Қосымшаның негізгі навигациясы |
| 📐 | **Үшбұрыш калькуляторы** | Үшбұрыштардың параметрлерін есептеу |
| 🎮 | **Угадай число ойыны** | Математикалық ойын-жаттығу |
| 📊 | **Прогрессиялар** | Арифметикалық және геометриялық прогрессиялармен жұмыс |
| 🎲 | **3D Фигуралар** | Үш өлшемді пішіндерді визуализациялау |

## 🚀 Орнату

### Талаптар

- Flutter SDK **3.11** немесе жоғары
- Dart SDK **3.11** немесе жоғары
- Android Studio / VS Code
- Android немесе iOS құрылғысы/эмуляторы

### Қадамдар

```bash
# Репозиторийді клондау
git clone https://github.com/yourusername/omnicalc.git
cd omnicalc

# Тәуелділіктерді орнату
flutter pub get

# Қосымшаны іске қосу
flutter run
```

### APK жинау

```bash
# Debug APK
flutter build apk --debug

# Release APK
flutter build apk --release
```

## 🎨 Дизайн жүйесі

OmniCalc келесі дизайн принциптерін қолданады:

- **Түстер палитрасы**: Күлгін-көк акценттермен қараңғы тақырып
- **Қаріп**: Outfit (Google Fonts)
- **Дөңгелектенген бұрыштар**: 16-20px border-radius
- **Анимациялар**: Тегіс ауысу эффектілері (300ms)
