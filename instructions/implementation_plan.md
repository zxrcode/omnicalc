#### [NEW] [triangle_screen.dart](file:///c:/Users/zxrd/AndroidStudioProjects/omnicalc/lib/screens/triangle_screen.dart)
**Покрывает задания A1, B1:**

- Вычисление площади (формула Герона: S = √(p(p-a)(p-b)(p-c)))

- Определение типа: сүйірбұрышты (остроугольный), тікбұрышты (прямоугольный), доғалбұрышты (тупоугольный)
- Визуализация треугольника на `CustomPainter`
- Использует **циклы** (для итерации по сторонам) и **условия** (проверка типа)

#### [NEW] [progressions_screen.dart](file:///c:/Users/zxrd/AndroidStudioProjects/omnicalc/lib/screens/progressions_screen.dart)
**Покрывает задания A2, C2:**
- Табы: «Геометриялық» / «Арифметикалық»
- **Геометрическая прогрессия**: ввод b₁, q, n → вычисление суммы Sn = b₁(qⁿ-1)/(q-1)
- **Арифметическая прогрессия**: ввод a₁, d, n → вычисление суммы Sn = n/2·(2a₁+(n-1)d), вывод n-го члена aₙ = a₁ + (n-1)d
- Вывод всех членов прогрессии (цикл for)
- Использует **циклы** (генерация членов) и **условия** (валидация)

#### [NEW] [shapes_3d_screen.dart](file:///c:/Users/zxrd/AndroidStudioProjects/omnicalc/lib/screens/shapes_3d_screen.dart)
**Покрывает задание C1:**
- Выбор фигуры: Сфера, Куб, Пирамида, Призма
- Ввод параметров, вычисление полной поверхности и объёма
- Ограничение количества запросов (пользователь задаёт лимит при входе)
- Счётчик оставшихся запросов
- Формулы:
  - Сфера: S=4πr², V=4/3πr³
  - Куб: S=6a², V=a³
  - Пирамида (правильная четырёхугольная): S=a²+2a·l, V=1/3·a²·h
  - Призма (правильная четырёхугольная): S=2a²+4ah, V=a²h
- Использует **циклы** (ведение истории) и **условия** (выбор фигуры, лимит)
